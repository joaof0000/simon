# simon

Simon Game
A simple, but captivating web version of the classic Simon game. This implementation uses HTML5, CSS3, and JavaScript to provide an engaging and interactive user experience. The game's goal is to follow an increasing sequence of flashing colors. The sequence gets longer each time the player completes the sequence correctly. The game continues until the player fails to follow the sequence.


Features
Six different sections each having distinct colors and sounds.
The game becomes progressively difficult as the sequence gets longer with each correct input.
Mobile responsive design, the game can be played on devices of different screen sizes.
Flashing animations to indicate active color in sequence.
Score tracking to keep up with the player's level.
How to Play
Click the "Start Game" button.
Watch and listen to the sequence presented.
Repeat the sequence by clicking on the correct color sections in the correct order.
If you complete the sequence correctly, a new step is added.
If you fail to follow the sequence correctly, it's game over, and you can restart.
The game continues until the player fails to match the sequence.
